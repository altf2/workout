# core_training.txt

= Mandatory Core Training Exercises

  * Tai chi waist turner ( 50-100 )
  * Isometric Stomach Flattener 
      Inhale, fill lungs. 7 sec breathe out - SSSS sounds.
  * The Vacuum
      Bend forward, exhale air from lunges, pull in and hold 6 sec ( 5-10 )

= Abdominal exercies for the front

  * The gut sucker
      Bend lightly, exhale air lunges, suck belly towards spine. Pump button max times ( 2-3 )
  * Standing Knee Raises
      Arms parallel floor, inhale raise knee, hold count 3, exahale - right leg ( max )
  * The Woodchopper
